# SavorIQ Web Frontend

The executive analytics dashboard for SavorIQ, built with Next.js.

## Overview

The web frontend provides high-level insights for restaurant group owners and executives:
1. **Executive Summary**: Aggregate performance across multiple locations.
2. **Deep Analytics**: Long-term sentiment trends and risk assessment for menu items.
3. **Admin Controls**: Managing restaurant tenants and global configuration.

## Key Components

- **`src/app/`**: Next.js App Router definitions.
- **`src/components/`**: Shared React components and UI elements.
- **`src/context/`**: React contexts for managing restaurant and data state.

## Local Development

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
